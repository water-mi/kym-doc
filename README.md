# 这里是匡院课程笔记分享站

![](https://img.shields.io/github/stars/kym-doc/doc?style=for-the-badge)

打算将这个网站建立成匡亚明学院本科学习过程中的一个笔记分享站。

> [!NOTE] 
> **什么是可以分享的？**
>
> 1. 某一门课的课程信息与大纲（分数占比、考核方式、涵盖内容等）
> 2. 某一门课的课程笔记（不求完全，因为一门课不可能所有的东西都值得记笔记）
> 3. 针对课程中某个知识点的补充说明或者思考

> [!WARNING] 
> **什么是不建议分享的？**
>
> 1. 课程的作业或者考试答案（如果觉得某个题目很有代表性，可以考虑分享一下题目和解题思路，但不建议直接分享答案）
> 2. 主观性的评价（不建议分享，可以私下讨论）

## 网站开源项目介绍

本站采用开源项目 Mkdocs 与 Material 进行构建，页面采用 Markdown 进行写作。您可以在下面找到它们的网站和非官方的中文化版本。如果您希望通过 Git 协作来完成贡献，您可能会用到这下面的内容。

Mkdocs: [官方](https://www.mkdocs.org/), [非官方中文](https://hellowac.github.io/mkdocs-docs-zh/)

Material: [官方](https://squidfunk.github.io/mkdocs-material/), [非官方中文](https://mkdocs.dbtgo.com/getting-started/)

## 如何反馈阅读中发现的问题或者疑惑

如果在阅读笔记的过程中发现了任何问题或者有任何疑惑，您可以通过 Github 仓库的 Issues 来反馈这些问题或者疑惑，或者直接通过社交媒体联系相应的文档维护者。

## 如何贡献自己的笔记

如果您有意向分享自己的笔记，有如下两种方式：

1. 将仓库 Clone 到本地，进行修改，并通过 Pull Request 来提交。您可以参考此页面“[添加笔记教程](contribute.md)”来了解文档的添加方式以及如何使用 Mkdocs 来预览修改后的效果。

2. 联系其他文档维护者代为提交，您应当向其提供 Markdown 源文件（如果该课程是首次添加，您需要额外提供一个 `about.md` 文件用于介绍该课程）或者已经整理好的 pdf 文件。
