## 课程感受

课程给分很好，实验里面写代码的部分基本上ai就能秒杀，但搭建网络环境什么的还挺麻烦的，实验手册也没有很清晰，不过给分很好。期中和期末分别考上下半学期的内容，其中期中考试有难度（笔者算错了一整道计算题，一定要多练练计算网络延迟的题目...），期末挺简单的，基本提前都能交卷。

关于听课：张渊老师的英文有点卡顿的感觉（没有不好的意思），课件也是英文，听着比较累。笔者基本没有听课，推荐直接看cs144即可，至少这个有字幕。

关于期末复习：课程肯定有部分内容是cs144没有提到的，全英的ppt看着也比较累还容易看不懂，笔者的焚诀是把每章ppt交给豆包让他生成详尽的复习笔记即可，~~这也是笔者没有笔记的理由~~。
